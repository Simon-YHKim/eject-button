Eject Button 给你一个体面的出口 — 一通由你自己触发、看起来非常真实的"假来电"。

■ 使用方法
点击 EJECT，手机立即响起。或设定计时（10秒 / 30秒 / 1分钟 / 自定义），让来电在最关键的时刻响起。把手机放在口袋里轻轻摇一摇，也能悄悄触发。

来电界面采用熟悉的通话 UI 风格，自然不突兀。接听后看着内置脚本提示讲几句，然后从容离开。无需尴尬告别，无需多余解释。

Eject Button 仅供您本人脱困使用，请勿用于欺骗、欺诈或伤害他人。

■ 主要功能
• 即时 / 定时 / 摇一摇 / 自定义延迟 触发
• 真实铃声 + 震动（使用手机默认）
• 可自定义来电人姓名 — 妈妈、上司、任何人
• 通话过程中显示对话脚本提示
• 通话历史记录
• 支持 7 种语言：中文、English、한국어、日本語、Español、हिन्दी

■ 适用场景
→ 商务饭局拖到三摊，无法体面退场
→ 让停不下来的初次约会画上句号
→ 没完没了的销售推销电话
→ 尴尬的家庭聚会
→ 不想继续的一对一谈话

不是恶搞。不是玩笑。这是面对真实社交压力的真实工具。
